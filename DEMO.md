# 🎬 Demo - Safety Content Creator

## 🚀 Szybki Start

### 1. Uruchomienie Aplikacji
```bash
cd SafetyContentCreator
python3 start.py
```

### 2. Otwórz Przeglądarkę
Aplikacja otworzy się automatycznie pod adresem: http://localhost:5000

## 🛠️ Funkcje Eksportu

Po wygenerowaniu posta dostępne są przyciski:

### 📥 Podstawowy Eksport
- **📋 Kopiuj Tekst**: Skopiuj treść do schowka
- **📄 Pobierz Obrazek**: Pobierz grafikę PNG 1080x1080
- **🔄 Nowy Post**: Wyczyść formularz

### 📱 Zaawansowany Eksport
- **📱 Instagram Story**: Format 1080x1920px
- **🖼️ Wszystkie Formaty**: Instagram, Facebook, LinkedIn
- **📅 Harmonogram**: Plan treści na miesiąc

## 💡 Przykładowe Wygenerowane Treści

### Post BHP (Przykład)
```
💡 Czy wiesz że...

Czy wiesz, że 85% wypadków przy pracy można było uniknąć?

Regularne szkolenia BHP to nie koszt, to inwestycja. Firmy z prawidłowo przeszkolonymi pracownikami notują 70% mniej wypadków przy pracy.

💬 Potrzebujesz pomocy z BHP i PPOŻ?
📧 Skontaktuj się z nami - jesteśmy ekspertami!
🎯 Bezpłatna konsultacja dla Twojej firmy

#InżynieriaBezpieczeństwa #BHP #PPOŻ #SzkoleniaBHP #BezpieczeństwoPracy
```

### Koncepcja Reelsa (Przykład)
```
🎬 5 sekund, które mogą uratować życie

📝 SCENARIUSZ:
0-2s: Pracownik wchodzi na plac budowy bez kasku
2-3s: STOP! Czerwony alarm na ekranie  
3-5s: Ten sam pracownik zakłada kask ochronny
5-8s: Tekst: "BHP to nie opcja, to konieczność"
8-10s: Logo firmy + "Bezpłatne szkolenia BHP"

🎵 MUZYKA: Energiczna, ale poważna - buduje napięcie

📱 HASHTAGI: #BHP #PPOŻ #SzkoleniaBHP #BezpieczeństwoPracy
```

## 📱 Test na iPadzie

### Kroki:
1. Uruchom aplikację na komputerze
2. Znajdź adres IP: `ifconfig | grep inet`
3. Na iPadzie wejdź na: `http://[IP]:5000`
4. Dodaj do ekranu głównego (Safari → Udostępnij → Dodaj do ekranu)

## 🎯 Najlepsze Praktyki Demo

### Do Pokazania Klientowi:
1. **Szybkość**: Post gotowy w 30 sekund
2. **Jakość**: Profesjonalne grafiki
